Cumplimiento

¿Es seguro para mi cuenta?

La primera pregunta que hay que hacerle a cualquier herramienta de LoL. Respuesta corta: sí. La larga:

  • Solo APIs oficiales de Riot. Live Client Data API, cliente de LoL (LCU), API web de Riot y Data Dragon. Nada de leer memoria, ni tráfico de red, ni repeticiones.
  • Solo información visible en pantalla. Todo lo que la app sabe de la partida podrías verlo tú mismo en el marcador. Sin tracking de cooldowns ni ultimates enemigos (prohibido por Riot desde marzo de 2025 — y aquí, regla innegociable del proyecto).
  • Local-first. No hay servidor, no hay cuentas, no hay telemetría. Tu historial y tus estadísticas viven en una base de datos en tu PC.
  • Código abierto. Puedes leer exactamente qué hace en GitHub.
Instalación

Cómo empezar

Instala

Descarga el setup para Windows y ejecútalo. SmartScreen avisará porque la app no está firmada: pulsa Más información → Ejecutar de todas formas. Se instala solo para tu usuario, sin permisos de administrador, y a partir de ahí se actualiza sola.

Consigue tu clave de Riot (gratis)

Para leer tu historial necesitas una clave personal de la API de Riot: regístrala una sola vez como "Personal App" en developer.riotgames.com y pégala en Ajustes. Las normas de Riot exigen que cada persona use la suya.

Juega

Deja la app abierta (un segundo monitor va perfecto). Detecta sola el cliente, el champ select y la partida: no hay que tocar nada más.

FAQ

Preguntas frecuentes

¿Me pueden banear por usarla?

La app se construyó con las normas de Riot para herramientas de terceros como reglas duras del proyecto: solo fuentes oficiales, solo información visible en pantalla y nada de tracking de cooldowns enemigos. No debería suponer ningún riesgo para tu cuenta — dicho esto, es un proyecto personal y se ofrece sin garantías.

¿Por qué Windows avisa al instalar?

Porque el instalador no está firmado digitalmente (los certificados de firma cuestan cientos de euros al año y esto es un proyecto gratuito). El aviso de SmartScreen es genérico para cualquier app sin firma; el código es público y puedes compilarla tú mismo desde GitHub si prefieres.

¿Es gratis? ¿Qué gano yo y qué ganas tú?

Es gratis y de código abierto para uso no comercial (licencia PolyForm Noncommercial). No hay anuncios ni recogida de datos: nació como herramienta personal y se comparte tal cual.

¿Qué datos usa y dónde acaban?

Tu partida en directo (la misma API local que usa el propio juego en el puerto 2999), el cliente de LoL para el champ select, y tu historial vía la API web de Riot con TU clave. Todo se guarda en una base de datos SQLite en tu PC. No hay servidor del proyecto: nada sale de tu máquina.

¿Por qué tengo que poner una clave de la API?

Riot prohíbe que las apps compartan una clave embebida: cada usuario debe registrar la suya (gratis, una vez). A cambio, tu historial va directo de Riot a tu PC sin intermediarios. Está en el roadmap un modo sin clave mediante un servicio propio con clave de producción de Riot.

¿Funciona en pantalla completa?

La app principal sí (segundo monitor o alt-tab). El overlay in-game opcional necesita LoL en modo ventana o sin bordes — la pantalla completa exclusiva no permite superponer ventanas.

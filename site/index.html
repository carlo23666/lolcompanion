Compliance

Is it safe for my account?

The first question to ask of any LoL tool. Short answer: yes. The long one:

  • Official Riot APIs only. Live Client Data API, the LoL client (LCU), Riot Web API and Data Dragon. No memory reading, no traffic sniffing, no replay parsing.
  • Only what's visible on your screen. Everything the app knows about a live game you could read off the scoreboard yourself. No enemy cooldown or ultimate tracking (banned by Riot since March 2025 — and a hard rule of this project).
  • Local-first. No server, no accounts, no telemetry. Your history and stats live in a database on your PC.
  • Open source. You can read exactly what it does on GitHub.
Setup

Get started

Install

Download the Windows setup and run it. SmartScreen will warn you because the app isn't code-signed: click More info → Run anyway. It installs per-user, no admin rights, and auto-updates from then on.

Get your free Riot API key

Reading your match history needs a personal Riot API key: register a "Personal App" once at developer.riotgames.com and paste the key in Settings. Riot's rules require everyone to use their own key.

Play

Leave the app open (a second monitor is perfect). It detects the client, champ select and the game by itself — nothing else to do.

Languages: the app is available in English and Spanish — pick your language in Settings (English is the default). The screenshots above happen to show the Spanish UI.

FAQ

Frequently asked questions

Can I get banned for using it?

The app was built with Riot's third-party rules as hard project constraints: official sources only, screen-visible information only, and no enemy cooldown tracking of any kind. It shouldn't pose any risk to your account — that said, it's a personal project provided without warranties.

Why does Windows warn me during install?

The installer isn't digitally signed (signing certificates cost hundreds of euros a year and this is a free project). The SmartScreen warning is generic for any unsigned app; the code is public and you can build it yourself from GitHub if you prefer.

Is it free? What's the catch?

Free and open source for noncommercial use (PolyForm Noncommercial license). No ads, no data collection: it started as a personal tool and is shared as-is.

What data does it use and where does it end up?

Your live game (the same local API the game itself exposes on port 2999), the LoL client for champ select, and your match history via the Riot Web API with YOUR key. Everything is stored in a SQLite database on your PC. There is no project server: nothing leaves your machine.

Why do I need to provide an API key?

Riot forbids apps from shipping a shared embedded key: each user registers their own (free, one time). In exchange, your history flows straight from Riot to your PC with no middleman. A keyless mode backed by a Riot production key is on the roadmap.

Does it work in fullscreen?

The main app does (second monitor or alt-tab). The optional in-game overlay needs LoL in windowed or borderless mode — exclusive fullscreen doesn't allow overlays.

Is the app in English?

Yes. The app ships in both English and Spanish — choose your language in Settings (English is the default; the screenshots above show the Spanish UI). Everything the app generates, including the coach's advice, follows your chosen language.
